# docs/superpowers/

This directory contains historical design documents from v1.1–v1.3 era, created using the superpowers brainstorming/planning workflow.

These documents reference features and architectures (bridge mode, bollard API, etc.) that may no longer exist in the current codebase (v1.5+). They are preserved as design history, not as current specifications.

For current specifications, see `docs/specs/v1.5/` and `docs/design.md`.
