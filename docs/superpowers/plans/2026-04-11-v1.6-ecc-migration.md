# vibepod v1.6: ECC Migration Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Move agent/skill content ownership from vibepod to ecc (everything-claude-code) via a cloned cache, add `--mode impl|review` axis, and keep `--template` for custom templates only.

**Architecture:** `vibepod init` clones ecc into `~/.config/vibepod/ecc-cache/`. Each language bundle (`templates-data/<lang>/<mode>/`) contains only `CLAUDE.md`, `settings.json`, and `vibepod-template.toml` with an `[ecc]` section listing which ecc files to pick. At `vibepod run`, vibepod assembles a staging dir (vibepod files + ecc-picked files) and mounts it into the container. Auto-refresh via background `git fetch` with copy-before-fetch to avoid races.

**Tech Stack:** Rust (existing vibepod CLI), TOML (existing parser), Docker (existing), git CLI (shell out), `include_dir!` (existing).

```rust
#[derive(Debug, Clone, serde::Serialize, serde::Deserialize)]
#[serde(deny_unknown_fields)]
pub struct EccConfig {
    #[serde(default = "default_ecc_repo")]
    pub repo: String,
    #[serde(default = "default_ecc_ref", rename = "ref")]
    pub r#ref: String,
    #[serde(default = "default_refresh_ttl")]
    pub refresh_ttl: String,
    #[serde(default = "default_auto_refresh")]
    pub auto_refresh: bool,
}

fn default_ecc_repo() -> String {
    "https://github.com/affaan-m/everything-claude-code".to_string()
}
fn default_ecc_ref() -> String { "main".to_string() }
fn default_refresh_ttl() -> String { "24h".to_string() }
fn default_auto_refresh() -> bool { true }

impl Default for EccConfig {
    fn default() -> Self {
        Self {
            repo: default_ecc_repo(),
            r#ref: default_ecc_ref(),
            refresh_ttl: default_refresh_ttl(),
            auto_refresh: default_auto_refresh(),
        }
    }
}

impl EccConfig {
    /// Parse `refresh_ttl` (e.g. "24h", "30m", "2h30m") into seconds.
    /// Returns Err if the string is not parseable.
    pub fn refresh_ttl_seconds(&self) -> u64 {
        parse_duration_to_seconds(&self.refresh_ttl).unwrap_or(24 * 60 * 60)
    }

    pub fn validate(&self) -> anyhow::Result<()> {
        parse_duration_to_seconds(&self.refresh_ttl)
            .ok_or_else(|| anyhow::anyhow!(
                "invalid [ecc] refresh_ttl '{}': expected format like '24h', '30m', '2h30m'",
                self.refresh_ttl
            ))?;
        if self.repo.is_empty() {
            anyhow::bail!("invalid [ecc] repo: must not be empty");
        }
        if self.r#ref.is_empty() {
            anyhow::bail!("invalid [ecc] ref: must not be empty");
        }
        Ok(())
    }
}

/// Parse "24h" / "30m" / "90s" / "2h30m" → total seconds. None if unparseable.
fn parse_duration_to_seconds(s: &str) -> Option<u64> {
    let s = s.trim();
    if s.is_empty() { return None; }
    let mut total: u64 = 0;
    let mut num: u64 = 0;
    let mut seen_digit = false;
    for c in s.chars() {
        if let Some(d) = c.to_digit(10) {
            num = num.checked_mul(10)?.checked_add(d as u64)?;
            seen_digit = true;
        } else {
            if !seen_digit { return None; }
            let multiplier = match c {
                's' => 1,
                'm' => 60,
                'h' => 60 * 60,
                'd' => 24 * 60 * 60,
                _ => return None,
            };
            total = total.checked_add(num.checked_mul(multiplier)?)?;
            num = 0;
            seen_digit = false;
        }
    }
    if seen_digit { return None; }  // trailing digits without unit
    Some(total)
}
```

```rust
/// Ensure `cache_dir(config_dir)` contains a clone of `cfg.repo` at `cfg.ref`.
/// If the directory doesn't exist, performs `git clone --depth 1 --branch <ref>`.
/// If the directory exists, returns Ok without touching git (use `fetch_latest`
/// for updates).
///
/// Idempotent: safe to call multiple times; a no-op if clone exists.
pub fn ensure_cloned(
    config_dir: &std::path::Path,
    cfg: &crate::config::EccConfig,
) -> Result<()> {
    let cache = cache_dir(config_dir);
    if cache.join(".git").is_dir() {
        return Ok(());
    }

    // Ensure parent exists
    if let Some(parent) = cache.parent() {
        std::fs::create_dir_all(parent)?;
    }

    // Remove any half-populated dir from a previous failed clone.
    if cache.exists() {
        std::fs::remove_dir_all(&cache)?;
    }

    let output = std::process::Command::new("git")
        .arg("clone")
        .arg("--depth").arg("1")
        .arg("--branch").arg(&cfg.r#ref)
        .arg(&cfg.repo)
        .arg(&cache)
        .output()
        .map_err(|e| anyhow::anyhow!("failed to spawn git: {e}"))?;

    if !output.status.success() {
        let stderr = String::from_utf8_lossy(&output.stderr);
        anyhow::bail!(
            "git clone failed ({}): {}",
            output.status, stderr.trim()
        );
    }
    Ok(())
}
```

```rust
/// Fetch and hard-reset the ecc cache to `cfg.ref`. Caller must ensure
/// `ensure_cloned` has been run first.
///
/// Errors if the cache directory is missing or the git operations fail.
pub fn fetch_latest(
    config_dir: &std::path::Path,
    cfg: &crate::config::EccConfig,
) -> Result<()> {
    let cache = cache_dir(config_dir);
    if !cache.join(".git").is_dir() {
        anyhow::bail!(
            "ecc cache not initialized at {}: run `vibepod init` first",
            cache.display()
        );
    }

    let run = |args: &[&str]| -> Result<()> {
        let output = std::process::Command::new("git")
            .current_dir(&cache)
            .args(args)
            .output()
            .map_err(|e| anyhow::anyhow!("failed to spawn git: {e}"))?;
        if !output.status.success() {
            anyhow::bail!(
                "git {} failed ({}): {}",
                args.join(" "),
                output.status,
                String::from_utf8_lossy(&output.stderr).trim()
            );
        }
        Ok(())
    };

    run(&["fetch", "--depth", "1", "origin", &cfg.r#ref])?;
    run(&["reset", "--hard", &format!("FETCH_HEAD")])?;
    Ok(())
}
```

```rust
/// Age of the ecc cache in seconds. Uses mtime of `.git/FETCH_HEAD`
/// (fetches update this) or `.git/HEAD` (fresh clone only) whichever
/// is newer. Returns None if cache doesn't exist.
pub fn cache_age_seconds(config_dir: &std::path::Path) -> Option<u64> {
    let cache = cache_dir(config_dir);
    let fetch_head = cache.join(".git/FETCH_HEAD");
    let head = cache.join(".git/HEAD");
    let candidate_mtime = [fetch_head, head]
        .iter()
        .filter_map(|p| p.metadata().ok())
        .filter_map(|m| m.modified().ok())
        .max()?;
    let now = std::time::SystemTime::now();
    now.duration_since(candidate_mtime).ok().map(|d| d.as_secs())
}
```

```rust
/// If `cfg.auto_refresh` is true and cache age > `refresh_ttl`,
/// spawn a detached background `git fetch + reset --hard` process.
/// Returns immediately — does not wait for the fetch.
///
/// Safe to call on every `vibepod run`: no-op when cache is fresh
/// or when auto_refresh is disabled.
///
/// CAUTION: this mutates the cache directory in-place. Callers that
/// need to read ecc files from the cache in this run MUST copy them
/// to the staging directory BEFORE calling this function.
pub fn maybe_background_refresh(
    config_dir: &std::path::Path,
    cfg: &crate::config::EccConfig,
) {
    if !cfg.auto_refresh {
        return;
    }
    let age = match cache_age_seconds(config_dir) {
        Some(a) => a,
        None => return,  // cache doesn't exist; init should handle this
    };
    if age < cfg.refresh_ttl_seconds() {
        return;
    }

    let cache = cache_dir(config_dir);
    let r#ref = cfg.r#ref.clone();
    // Fire-and-forget: ignore errors, stderr swallowed. The next run
    // will see either the new state (success) or the old state (failure),
    // both of which are safe.
    let _ = std::process::Command::new("sh")
        .arg("-c")
        .arg(format!(
            "cd {cache:?} && git fetch --depth 1 origin {ref_arg} \
             && git reset --hard FETCH_HEAD",
            cache = cache,
            ref_arg = shell_escape(&r#ref),
        ))
        .stdout(std::process::Stdio::null())
        .stderr(std::process::Stdio::null())
        .spawn();
}

fn shell_escape(s: &str) -> String {
    format!("'{}'", s.replace('\'', r"'\''"))
}
```

```rust
/// `vibepod template status`: print ecc-cache state.
pub fn status() -> Result<()> {
    let config_dir = config::default_config_dir()?;
    let cache = crate::ecc::cache_dir(&config_dir);

    let unified = config::load_unified(&config_dir)?;
    let ecc_cfg = unified.ecc.unwrap_or_default();

    println!("ecc repo:         {}", ecc_cfg.repo);
    println!("configured ref:   {}", ecc_cfg.r#ref);
    println!("refresh_ttl:      {}", ecc_cfg.refresh_ttl);
    println!("auto_refresh:     {}", ecc_cfg.auto_refresh);
    println!("cache dir:        {}", cache.display());

    if !cache.join(".git").is_dir() {
        println!("cache status:     not initialized — run `vibepod init`");
        return Ok(());
    }

    // Current commit
    let commit = std::process::Command::new("git")
        .current_dir(&cache)
        .args(["rev-parse", "--short", "HEAD"])
        .output()
        .ok()
        .and_then(|o| {
            if o.status.success() {
                String::from_utf8(o.stdout).ok()
            } else {
                None
            }
        })
        .map(|s| s.trim().to_string())
        .unwrap_or_else(|| "unknown".to_string());
    println!("current commit:   {commit}");

    if let Some(age) = crate::ecc::cache_age_seconds(&config_dir) {
        let hours = age / 3600;
        let minutes = (age % 3600) / 60;
        println!("last updated:     {hours}h{minutes}m ago");
    }

    Ok(())
}
```

```rust
/// `vibepod template update [--ref <ref>]`: blocking fetch + reset of
/// the ecc cache. Optionally override the configured ref.
pub fn update(ref_override: Option<&str>) -> Result<()> {
    let config_dir = config::default_config_dir()?;
    let unified = config::load_unified(&config_dir)?;
    let mut ecc_cfg = unified.ecc.unwrap_or_default();
    if let Some(r) = ref_override {
        ecc_cfg.r#ref = r.to_string();
    }
    ecc_cfg.validate()?;

    let cache = crate::ecc::cache_dir(&config_dir);
    if !cache.join(".git").is_dir() {

            "ecc cache not initialized at {}: run `vibepod init` first",
            cache.display()
        );
    }

    println!("Fetching ecc ref '{}' into {}...", ecc_cfg.r#ref, cache.display());
    crate::ecc::fetch_latest(&config_dir, &ecc_cfg)?;
    println!("Updated.");
    Ok(())
}
```

```rust
/// Copy selected ecc files from `cache_dir(config_dir)` to the staging
/// directory. Skills keep their directory structure
/// (`skills/<name>/SKILL.md` → `.claude/skills/<name>/SKILL.md`).
/// Agents are flat files (`agents/<n>.md` → `.claude/agents/<n>.md`).
///
/// Fails fast if any listed file is missing in the cache.
pub fn stage_files(
    config_dir: &std::path::Path,
    runtime_dir: &std::path::Path,
    selection: &crate::cli::run::template::EccSelection,
) -> Result<()> {
    let cache = cache_dir(config_dir);
    let staging = staging_dir(runtime_dir);

    let staging_skills = staging.join(".claude/skills");
    let staging_agents = staging.join(".claude/agents");

    for rel in &selection.skills {
        let src = cache.join(rel);
        if !src.is_file() {
            anyhow::bail!(
                "ecc skill not found in cache: '{}' (expected at {})",
                rel, src.display()
            );
        }
        // rel is e.g. "skills/rust-patterns/SKILL.md"
        // strip leading "skills/" and place under staging_skills
        let stripped = rel.strip_prefix("skills/").ok_or_else(|| {
            anyhow::anyhow!(
                "ecc skill path '{}' must start with 'skills/'", rel
            )
        })?;
        let dest = staging_skills.join(stripped);
        if let Some(parent) = dest.parent() {
            std::fs::create_dir_all(parent)?;
        }
        std::fs::copy(&src, &dest)
            .map_err(|e| anyhow::anyhow!("failed to copy {}: {}", rel, e))?;
    }

    for rel in &selection.agents {
        let src = cache.join(rel);
        if !src.is_file() {
            anyhow::bail!(
                "ecc agent not found in cache: '{}' (expected at {})",
                rel, src.display()
            );
        }
        let stripped = rel.strip_prefix("agents/").ok_or_else(|| {
            anyhow::anyhow!(
                "ecc agent path '{}' must start with 'agents/'", rel
            )
        })?;
        let dest = staging_agents.join(stripped);
        if let Some(parent) = dest.parent() {
            std::fs::create_dir_all(parent)?;
        }
        std::fs::copy(&src, &dest)
            .map_err(|e| anyhow::anyhow!("failed to copy {}: {}", rel, e))?;
    }

    Ok(())
}
```

```rust
/// Resolve (lang, mode) to an embedded template directory name relative
/// to `templates-data/`. Returns None when the pair has no official
/// template (caller falls back to host template for `(None, Impl)` or

pub fn resolve_official_template_dir(
```

```rust
/// Assemble the staging directory used as the container's `/home/vibepod/.claude/`
/// mount source. Combines:
///   1. All files from `template_dir` (CLAUDE.md, settings.json, etc.),
///      copied as-is into the staging root.
///   2. Files selected by `template_dir/vibepod-template.toml`'s [ecc]
///      section, copied from the ecc-cache into staging's `.claude/skills/`
///      and `.claude/agents/`.
///
/// Fails fast if any [ecc] file is missing from the cache.
///
/// Test-accessible name: call sites in prepare_context use this by a
/// private helper wrapping this; tests call this directly.
pub fn assemble_staging_for_test(
    config_dir: &std::path::Path,
    runtime_dir: &std::path::Path,
    template_dir: &std::path::Path,
) -> anyhow::Result<()> {
    let staging = crate::ecc::staging_dir(runtime_dir);
    if staging.exists() {
        std::fs::remove_dir_all(&staging)?;
    }
    std::fs::create_dir_all(&staging)?;

    // 1. Copy template_dir contents into staging root
    copy_dir_contents(template_dir, &staging)?;

    // 2. Read vibepod-template.toml and stage ecc files
    let meta_path = template_dir.join("vibepod-template.toml");
    if meta_path.is_file() {
        let meta = crate::cli::run::template::parse_template_metadata(&meta_path)?;
        if !meta.ecc.skills.is_empty() || !meta.ecc.agents.is_empty() {
            crate::ecc::stage_files(config_dir, runtime_dir, &meta.ecc)?;
        }
    }
    Ok(())
}

fn copy_dir_contents(src: &std::path::Path, dst: &std::path::Path) -> anyhow::Result<()> {
    for entry in std::fs::read_dir(src)? {
        let entry = entry?;
        let path = entry.path();
        let rel = path.strip_prefix(src).unwrap();
        let target = dst.join(rel);
        if path.is_dir() {
            std::fs::create_dir_all(&target)?;
            copy_dir_contents(&path, &target)?;
        } else {
            if let Some(p) = target.parent() { std::fs::create_dir_all(p)?; }
            std::fs::copy(&path, &target)?;
        }
    }
    Ok(())
}
```

After assembly is complete, fire-and-forget `ecc::maybe_background_refresh(&config_dir, &ecc_cfg)`.
